Topics and content

There are nine broad topics in the module, which are as follows:

  1. Introductory concepts
  2. Authentication
  3. Password cracking
  4. Cryptography
  5. Public key cryptography (Mathematics)
  6. Digital certificates
  7. Secure email
  8. Authentication protocols
  9. Virtualisation
